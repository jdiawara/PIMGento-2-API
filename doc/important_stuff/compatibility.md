# Compatibility with...

Akeneo
------

* PIMGento2 (API) is compatible with Akeneo PIM >= 2.0 (CE & EE)

Magento2
--------

* PIMGento2 (API) is compatible with Magento 2 following this schema:

| Magento Version | PIMGento2 (API) Version |
|-----------------|-------------------------|
| >= 2.1.x        | master                  |
